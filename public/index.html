why use this

Built for developers, not browsers.

Drop-in JSON API that abstracts scraping complexity — so you can build bots, apps, and automation without touching a single HTML parser.

Zero auth

No API keys, no registration, no rate-limit headers. Just HTTP GET.

JSON-first

Proxy download

Unified /api/dl endpoint auto-detects PNG/JPG/WebP and handles Referer headers.

3 sources, 1 API

Wallhaven, Steam Workshop, and WallpaperAccess unified behind one predictable interface.

4 endpoints

Everything you need, nothing you don't.

Each endpoint is a clean JSON contract. Hit it with curl, fetch, axios — anything that speaks HTTP.

GET
/api/wallhaven

Full-text search across wallhaven.cc — the largest free wallpaper archive. Returns thumbs + CDN URLs + proxy links.

q search limit 1–64 page 1..n purity 100 sort relevance|date|views atleast 1920x1080 format urls
/api/wallhaven?q=anime&limit=3&format=urls
GET
/api/dl

Unified direct-download endpoint. Auto-detects PNG / JPG / WebP, sets correct Referer, streams the binary back.

source wallhaven|wallpaperaccess id wallpaper id url passthrough
/api/dl?source=wallhaven&id=j37zv5
GET
/api/workshop

Browse Steam Workshop for Wallpaper Engine (appid 431960). Real titles, authors, preview URLs. Binary downloads redirect to source — Steam requires a subscription.

q text filter sort trend|newest|rating|popular page limit
/api/workshop?sort=trend&limit=10
GET
/api/wallpaperaccess

Direct image download by numeric ID. Search is gated by Cloudflare on wallpaperaccess.com — grab IDs from their site, fetch here.

id numeric id (required)
/api/wallpaperaccess?id=12578
response shape

What you get back.

Every endpoint returns clean JSON with deduped media, totals, and a cached flag. No scraping the source HTML on your end.

drop-in code

Use it in 3 lines.

No SDK. No npm install. Just fetch.

JavaScript

// Fetch 3 anime wallpapers as URLs const urls = await fetch( '/api/wallhaven?q=anime&limit=3&format=urls' ).then(r => r.json()); // Render in <img> urls.forEach(u => { document.body.innerHTML += `<img src="${u}">`; });

Python

import requests # Get a flat URL list urls = requests.get( "https://zade-wallpaper-api.vercel.app/api/wallhaven", params={"q": "anime", "limit": 3, "format": "urls"} ).json() for url in urls: print(url)
