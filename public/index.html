// endpoints

Six routes. Zero friction.

GET /api/wallhaven

Curated wallpapers from Wallhaven. Filter by query, category, purity, sorting. Returns full metadata + multiple CDN sizes.

q=string page=int limit=int sorting=string categories=int
GET /api/wallpaperaccess

Tagged wallpaper collections. Categories, popular tags, full-size images via proxy.

tag=string category=string page=int limit=int
GET /api/workshop

Steam Workshop Wallpaper Engine items. Animated + static from creator community.

query=string appid=int page=int limit=int
GET /api/dl

Browser-direct download proxy. Pass any image URL — server streams bytes back.

url=string

JavaScript
const res = await fetch('/api/wallhaven?q=nature&limit=6'); const data = await res.json(); data.results.forEach(wp => { console.log(wp.url); }); // direct download window.location.href = `/api/dl?url=${encodeURIComponent(wp.url)}`;
Python
import requests r = requests.get( "https://zade-wallpaper-api.vercel.app/api/wallhaven", params={"q": "nature", "limit": 6} ) data = r.json() for wp in data["results"]: print(wp["url"])
