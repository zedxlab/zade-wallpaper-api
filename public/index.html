Endpoints

Everything you need, nothing you don't

Six clean routes. Each returns structured JSON with consistent fields across sources.

/api/wallhaven Wallhaven.cc

Curated high-res wallpapers from Wallhaven. Filter by query, category, purity, sorting. Returns full image metadata + multiple CDN sizes.

q=string page=int limit=int sorting=string categories=int
/api/wallpaperaccess WallpaperAccess

Browse tagged wallpaper collections. Categories, popular tags, full-size images via proxy. Great for themed galleries.

tag=string category=string page=int limit=int
/api/workshop Steam Workshop

Wallpaper Engine Workshop items — animated and static wallpapers from Steam's creator community. Includes preview URLs and item metadata.

query=string appid=int page=int limit=int
/api/dl Proxy

Browser-direct download proxy. Pass any image URL — server fetches and streams bytes back so the browser saves the file directly.

url=string
Examples

Request · Response · Code

Copy-paste ready. No SDKs, no wrappers, no auth tokens.

# Fetch 6 nature wallpapers, sorted by favorites
GET https://zade-wallpaper-api.vercel.app/api/wallhaven?q=nature&limit=6&sorting=favorites

# Browse anime-tagged wallpapers
GET https://zade-wallpaper-api.vercel.app/api/wallpaperaccess?tag=anime&limit=8

# Search Steam Workshop for live wallpapers
GET https://zade-wallpaper-api.vercel.app/api/workshop?query=scenery&limit=5

# Direct-download proxy
GET https://zade-wallpaper-api.vercel.app/api/dl?url=<image_url>

// Fetch 6 nature wallpapers
const res = await fetch(
  'https://zade-wallpaper-api.vercel.app/api/wallhaven?q=nature&limit=6'
);
const data = await res.json();

// data.results[0].cdn_url — direct image link
// data.results[0].thumb_url — preview thumbnail

data.results.forEach(wp => {
  console.log(`${wp.filename} — ${wp.width}×${wp.height}`);
});

// Direct browser download via /api/dl proxy:
window.location.href =
  `/api/dl?url=${encodeURIComponent(data.results[0].cdn_url)}`;
# pip install requests
import requests

r = requests.get(
    "https://zade-wallpaper-api.vercel.app/api/wallhaven",
    params={"q": "nature", "limit": 6}
)
data = r.json()

for wp in data["results"]:
    print(f"{wp['filename']} — {wp['width']}×{wp['height']}")

import urllib.parse
