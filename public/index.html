API Reference

Four endpoints.
Zero complexity.

No API keys. No rate limits. No SDK. Just hit the URL, get JSON (or image bytes).

GET /api/wallhaven wallhaven.cc

Full-text search across Wallhaven's free wallpaper library. Returns rich metadata, thumbnails, and direct CDN URLs. Supports purity filters, sort modes, and resolution minimums.

Parameters
q page limit purity sort atleast format=urls
Example request
curl "https://zade-wallpaper-api.vercel.app/api/wallhaven?q=anime%20sunset&limit=5&purity=100"
GET /api/wallpaperaccess direct

Direct image download by numeric ID. Search is gated by Cloudflare, but CDN serves full-res JPG/PNG directly. Best for batch downloads with known IDs.

Parameters
id *
Example request
curl "https://zade-wallpaper-api.vercel.app/api/wallpaperaccess?id=12578" -o wallpaper.jpg
GET /api/workshop steam

Browse Steam Workshop wallpapers for Wallpaper Engine (appid 431960). Returns titles, authors, preview URLs, and Steam page links. Binary wallpapers require subscription.

Parameters
q sort page limit
Example request
curl "https://zade-wallpaper-api.vercel.app/api/workshop?sort=trend&limit=10"
GET /api/dl proxy

Unified image downloader. Pass source + id to auto-detect PNG/JPG/WebP, or pass a raw url for passthrough. Adds proper Referer headers and handles 404 fallbacks.

Parameters
source * id url
Example request
curl "https://zade-wallpaper-api.vercel.app/api/dl?source=wallhaven&id=j37zv5" -o image.png
